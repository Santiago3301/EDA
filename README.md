# EDA
Estructuras de Datos y Algoritmos - UNSL 2025

- main.c: programa principal para ver las demostraciones de las distintas Estructuras

Como usar:

1. Ejecutar main.c:
    - gcc main.c -o main
    - ./main

2. Seleccionar el tipo de sistema:
    - 1: Windows
    - 2: Linux

3. Seleccionar la categoria de las estructuras desde el menu
